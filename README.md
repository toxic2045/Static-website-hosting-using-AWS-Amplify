## How It Works

1. The website code is stored in a GitHub repository.
2. AWS Amplify connects to the GitHub repo.
3. Every push to the main branch triggers an automatic deployment.
4. The site is hosted on a public Amplify URL or a custom domain.

## Getting Started

### Prerequisites

- AWS account
- GitHub account
- Basic knowledge of HTML, CSS, and JavaScript

### Local Setup

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

Open `index.html` in your browser or use a local server.

### Deployment Steps

1. Push the project to GitHub.
2. Open AWS Amplify Console.
3. Connect the GitHub repository.
4. Select the branch to deploy.
5. Deploy and access the live website.
